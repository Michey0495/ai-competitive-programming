## 14. MCP接続方法

```
Claude DesktopでMCPサーバー接続する方法（1分でできる）

claude_desktop_config.jsonに追加:
{
  "mcpServers": {
    "ai-roast": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://ai-roast.ezoai.jp/api/mcp"]
    }
  }
}

これでClaude Desktopから「ローストして」が使える
```
